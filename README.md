# interactive_sales_dashboard-
# Interactive Sales Dashboard

## Project Overview

This project is an Interactive Sales Dashboard created using Python, Seaborn, Matplotlib, and Plotly.

The dashboard analyzes sales data and visualizes important business insights such as:

- Product performance
- Monthly sales trends
- Price distribution
- Correlation analysis
- Interactive sales visualization

---

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

# Visualizations Included

## 1. Bar Chart
Shows total sales by product.

## 2. Line Chart
Displays monthly sales trends over time.

## 3. Box Plot
Analyzes product price distribution.

## 4. Violin Plot
Shows detailed distribution and density of prices.

## 5. Heatmap
Displays correlations between numerical columns.

## 6. Dashboard Layout
Combines multiple charts into a single dashboard.

## 7. Interactive Dashboard
Interactive Plotly visualization with hover effects.
